== Installation ==

To install this theme, 
	1.	Log in to admin panel
	2.	Select “Appearance” -> “Themes” from the side menu.
	3.	Press “Add New” button
	4.	Type in Simple Corp in Search form and hit Enter.
	5.	Hover on the thumbnail and press “Install” -> “Activate”
